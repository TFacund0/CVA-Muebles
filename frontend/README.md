# CVA Muebles — Frontend

Frontend en Next.js (App Router) del proyecto CVA Muebles. Consume la API REST del backend en `/api/v1/*`.

Para arquitectura general del proyecto, estructura de carpetas y guía de instalación completa, ver el [README principal](../README.md) en la raíz del repositorio.

## Desarrollo local

```bash
cp .env.local.example .env.local   # ajustar NEXT_PUBLIC_API_URL
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).
